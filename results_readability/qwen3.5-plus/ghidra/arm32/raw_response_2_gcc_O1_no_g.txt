# Decompilation Readability Assessment

## === LEVEL 1: LEXICAL CLARITY (30% weight) ===

**1. Variable Naming Quality:** Score: 2/10  
Justification: All variables use generic names (`param_1`, `iVar1`, `uVar1`, `piVar2`) with zero semantic meaning. Original source had meaningful names like `sum`, `len`, `current`, `head` that are completely lost.

**2. Function Naming Quality:** Score: 8/10  
Justification: Most function names preserved from source (`double_value`, `process_char`, `array_1d_stack`, `struct_simple`). This is the decompiler's strongest aspect—action-oriented names clearly reflect purpose.

**3. Literal Clarity:** Score: 3/10  
Justification: Magic numbers everywhere without explanation (e.g., `0x41`, `0x1a`, `0x20` in `process_char`; `0x3fc00000` for float constants). Test functions show hardcoded expected values instead of actual computations.

**4. Noise Reduction:** Score: 5/10  
Justification: Compiler runtime functions included (`__muldf3`, `__aeabi_dadd`, `__addsf3`) mixing with user code. Stack canary checks (`__stack_chk_fail()`) at function ends. Unreachable block warnings present.

**6. Function Decomposition:** Score: 7/10

**7. Code Linearization:** Score: 6/10

**8. Redundancy Elimination:** Score: 5/10

## === LEVEL 3: TYPE SYSTEM FIDELITY (20% weight) ===

**9. Type Precision:** Score: 3/10  
Justification: Severe type degradation: `char`→`uint` in `process_char`, `short`→`int` in `process_short`, `long long` split into two `int` params in `process_ll`. Float/double functions return `void` instead of proper types. Pointer types become `undefined4*`.

**10. Composite Type Recovery:** Score: 2/10  
Justification: Complete failure—structs represented as `int*` with array indexing (`param_1[0]`, `param_1[1]`). No struct/union/enum definitions recovered. `Point3D`, `Rect`, `Widget` all lost.

**11. Type Safety:** Score: 3/10  
Justification: Many unsafe casts throughout. Pointer arithmetic on `int*` for struct field access. Function pointers become `code*` or `undefined4*`. Void pointer conversions without proper typing.
